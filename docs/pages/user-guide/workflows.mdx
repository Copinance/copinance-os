# Workflows

Copinance OS supports multiple workflow types for market research.

## Static Instrument Workflows

Static workflows follow a predefined analysis pipeline with consistent steps.

### Equity Workflow

What it does:
1. Fetches instrument information
2. Gets the current quote
3. Retrieves historical price data
4. Gets equity fundamentals
5. Calculates key metrics and trends
6. Generates a summary

Example:

```bash
copinance analyze market equity AAPL --timeframe mid_term
```

### Options Workflow

What it does:
1. Fetches the current underlying quote
2. Loads the options chain for the selected expiration
3. Summarizes open interest, volume, and implied volatility
4. Highlights the at-the-money contract

Example:

```bash
copinance analyze market options AAPL --expiration 2026-06-19
```

## Agentic Workflows

Agentic workflows use AI to dynamically analyze market instruments or the market based on your questions. The agent can use tools (quotes, historical data, options chain, fundamentals, macro indicators) to answer.

**Equity:**
```bash
copinance ask -q "What are the key financial risks?" -i AAPL
```

**Options:** Ask about specific stock options data; use `--market-type options` and optionally `--expiration` and `--side` so the agent gets options context and can fetch the options chain.
```bash
copinance ask -q "How is the options chain positioned?" -i AAPL --market-type options
copinance ask -q "What's the put/call open interest for AAPL?" -i AAPL --market-type options --expiration 2026-06-19
copinance ask -q "What are the most active calls?" -i AAPL --market-type options --side call
```

**Market-wide:**
```bash
copinance ask -q "What is the current market sentiment?" -m SPY
```

See [CLI Reference - Ask](cli#ask-agentic) for all options. Use `--show-prompt` to display the system and user prompt used for the question before the answer.

## Timeframes

Both static and agentic workflows adapt to the timeframe:

### Short-term
- **Focus**: Recent price movements, short-dated options context, technical indicators
- **Data**: Last 30 days

### Mid-term
- **Focus**: Quarterly fundamentals, price trends
- **Data**: Last 6 months

### Long-term
- **Focus**: Annual fundamentals, long-term trends
- **Data**: Last 2 years

## One-off Execution

All workflow runs are one-off: `copinance analyze` and `copinance ask` run workflows without persisting jobs. Each run saves results to `.copinance/results/v2/` as versioned JSON envelopes. Profiles and cached tool data also live under versioned subdirectories in `.copinance`.

### Example Workflows

```bash
# Equity analysis
copinance analyze market equity AAPL --timeframe mid_term

# Options analysis
copinance analyze market options AAPL --expiration 2026-06-19

# Macro regime
copinance analyze macro --market-index SPY

# Agentic Q&A (equity)
copinance ask -q "What are the key risks?" -i AAPL

# Agentic Q&A (options)
copinance ask -q "Summarize AAPL options for June 2026" -i AAPL --market-type options --expiration 2026-06-19
```

## Macro Regime Workflow

The macro regime workflow combines market regime indicators with macroeconomic data to provide a comprehensive view of market conditions.

### Quick Start

```bash
copinance analyze macro
```

### Customization

```bash
copinance analyze macro --market-index QQQ --lookback-days 90
copinance analyze macro --no-include-vix --no-include-market-breadth --no-include-sector-rotation
```

### Data Sources

- Macro indicators prefer FRED API when configured
- Fallback proxies use yfinance where needed
- Market regime indicators use yfinance market data
