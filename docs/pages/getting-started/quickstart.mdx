# Quick Start

Get started with Copinance OS in 5 minutes.

## Your First Analysis

### 1. Search for an Instrument

```bash
copinance market search "Apple"
```

This shows matching market instruments and their symbols.

### 2. Run an Equity Workflow

```bash
copinance analyze market equity AAPL --timeframe mid_term
```

This performs a comprehensive equity workflow with:
- Current market quote
- Historical price data
- Fundamental analysis
- Key metrics and trends

### 3. Check a Live Quote

```bash
copinance market quote AAPL
```

This returns the latest quote snapshot for the selected instrument.

### 4. Explore the Options Market

```bash
copinance market options AAPL --expiration 2026-06-19
```

This fetches an options chain snapshot with contract prices, implied volatility, open interest, and volume.

### 5. Ask a Question

```bash
copinance ask "What is the current price and P/E ratio?" --instrument AAPL
copinance ask "How is the options chain positioned?" --instrument AAPL --market-type options
```

The AI will analyze the selected market context using available data tools.

## Working with Profiles

Profiles let you customize analysis output based on your financial literacy level.

### Automatic Profile Prompting

When you run `analyze` or `ask` without a profile, the system will automatically prompt you to set your financial literacy level for more personalized results. You can:
- Choose to create a profile with your literacy level (`beginner`, `intermediate`, or `advanced`)
- Decline and continue with default settings

### Create a Profile Manually

```bash
copinance profile create --literacy intermediate --name "My Profile"
```

### Use Profile in Analysis

```bash
copinance analyze market equity AAPL --profile-id <profile-id>
```

Or simply use your current profile (set with `profile set-current`).

## Common Workflows

### Quick Analysis

```bash
# Static equity workflow
copinance analyze market equity AAPL --timeframe mid_term

# Static options workflow
copinance analyze market options AAPL --expiration 2026-06-19

# Agentic workflow (equity)
copinance ask -q "Analyze this equity for me" -i AAPL

# Agentic workflow (options)
copinance ask -q "What's the put/call open interest?" -i AAPL --market-type options --expiration 2026-06-19
```

### Data and Cache

All CLI data is stored under `.copinance` using versioned subdirectories such as `data/v2`, `cache/v2`, `results/v2`, and `state/v2`. Use `copinance cache info` to see the active cache location and `copinance cache clear` to clear cached data.

## Macro Regime Dashboard

Get a comprehensive view of market conditions and macroeconomic indicators:

```bash
# Full analysis with defaults
copinance analyze macro

# Custom index and lookback period
copinance analyze macro --market-index QQQ --lookback-days 180
```

See [Workflows](../user-guide/workflows#macro-regime-workflow) for all customization options.
